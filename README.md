# Система много-зонного полива (ПР-231, Команда №2)

## 1. Назначение проекта
Данный проект представляет собой прототип автоматизированной системы полива для теплиц на 3 зоны. 
Система управляет клапанами и насосом на основе показателей влажности почвы и временных расписаний, обеспечивая безопасность оборудования (защита от сухого хода бака).

## 3. MVP (Минимально жизнеспособный продукт)
* Чтение данных о влажности с 3-х зон с помощью аналоговых датчиков.
* Управление 4-мя реле (3 клапана зон + 1 основной насос).
* Система защиты: блокировка насоса при пустом баке и визуализация статуса на OLED-дисплее.

## 4. Как запустить
Проект разработан для симулятора Wokwi.
1. Скопируйте содержимое репозитория.
2. Откройте проект в Wokwi (web или VS Code).
3. Симуляция использует файл diagram.json для сборки схемы и main.py для логики на MicroPython.

## 5. Структура репозитория
* `main.py` — основной файл программы (скелет проекта).
* `diagram.json` — конфигурация компонентов в Wokwi.
* `docs/architecture.md` — подробное описание аппаратной части и интерфейсов.
* `README.md` — общая информация о проекте (этот файл).
