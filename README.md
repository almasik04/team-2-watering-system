# Система много-зонного полива (ПР-231, Команда №2)

## 1. Назначение проекта
[span_0](start_span)Данный проект представляет собой прототип автоматизированной системы полива для теплиц на 3 зоны[span_0](end_span). [span_1](start_span)[span_2](start_span)Система управляет клапанами и насосом на основе показателей влажности почвы и временных расписаний, обеспечивая безопасность оборудования (защита от сухого хода бака)[span_1](end_span)[span_2](end_span).

## 3. MVP (Минимально жизнеспособный продукт)
* [span_3](start_span)Чтение данных о влажности с 3-х зон с помощью аналоговых датчиков[span_3](end_span).
* [span_4](start_span)Управление 4-мя реле (3 клапана зон + 1 основной насос)[span_4](end_span).
* [span_5](start_span)Система защиты: блокировка насоса при пустом баке и визуализация статуса на OLED-дисплее[span_5](end_span).

## 4. Как запустить
Проект разработан для симулятора Wokwi.
1. Скопируйте содержимое репозитория.
2. Откройте проект в Wokwi (web или VS Code).
3. [span_6](start_span)Симуляция использует файл diagram.json для сборки схемы и main.py для логики на MicroPython[span_6](end_span).
